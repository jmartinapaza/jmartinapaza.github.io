Intro a CSS JM

Definición

Es un lenguaje usado para mejorar la presentación de un sitio web escrito en, por ejemplo, HTML5, haciendolo más accesible con colores, fuentes y "layouts" ó capas. Se crea entre el título y el cuerpo del documento HTML

Seleccionar por la etiqueta

Es importante saber que al seleccionar todos los elementos con la misma etiqueta, esta se escriba sin signo alguno. Aquí todos los h1 están en verde

Seleccionar por id

Si se quiere alterar un solo elemento podemos añadir una id, que se pone antes de cerrar la primer etiqueta (si hay dos). Para identificarla en "style" ponemos una almohadilla (#) antes del id. Se tiene que escribir el id con palabras exactas, ya que el ordernador no reconoce la misma palabra en mayúsculas y minúsculas, a su vez, no reconoce espacios vaciós, por lo que usamos guiones

Seleccionar por clase

Si queremos alterar varios elementos podemos asociarlos en un grupo en vez de tener un id único para cada uno, aparte de eso, se comporta de la misma manera que id excepto que hay que poner un punto (.) en vez de una almohadilla (#)

El texto en cursiva tiene un color diferente al texto en negrita, demostrando la propiedad de las clases
