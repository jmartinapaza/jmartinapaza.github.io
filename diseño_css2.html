Última modificación: 9 de diciembre de 2021

Esta etiqueta indica dónde se situan los elementos en varias maneras, teniendo variantes

1. Absoluto

Localiza al elemento dentro del borde de la página web, a no ser que este subordinado a otro

Este texto se situa 50 píxeles a la derecha y 75 hacia el techo de la página web

2. Relativo

Mientras, el relativo altera la posición estándar del elemento

Este texto puede hasta "meterse" dentro de la página, ocultando parte de este (valor de -50px)

3. Fijado

Similar al absoluto, pero que se incrusta en la página, no desapareciendo al desplazarse por el sitio web, aqui esta en el centro y suelo de la página web

Este texto se mantiene en su lugar de manera permanente

4. El z-index

Siendo solo aplicable a elementos con la etiqueta position (y otros), el z-index marca el orden de superponer varios elementos

En esta imagen hay un texto que se superpone y otro que se subordina

Este texto se superpone al logo

Mientras este texto se subordina al logo

El que superpone tiene una anchura de 150px y se eleva 50px y 70px a la izquierda,
mientras el que se subordina tiene 200 píxeles de ancho, se eleva 100px y va a la izquierda por 15%
