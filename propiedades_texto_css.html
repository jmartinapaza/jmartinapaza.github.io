Propiedades de texto en CSS

1. Fuentes de letras

Es un conjunto de signos de tamaño y tipo específico ilustrado por un procesador de textos
Para decirle al ordenador que tipo deberiamos usar, están la propiedad "font-family", esta a su vez se divide en dos

  • Genérico: En este caso el ordenador escoge la fuente que primero atribuye, como "sans-serif"
  • Específico: Y aquí se manda que el ordenador utilize la fuente escrita, por ejemplo, si queremos usar Helvetica, ponemos ""Helvetica", sans-serif"

Tamaño de fuente (font-size)

Aunque haya multiples maneras de escribirlas, nos quedaremos con dos:

  • En píxeles (px), útil para medidas precisas
  • En "em", con un número atrás siendo el multiplicador de la letra más pequeña

Este texto esta a tamaño 1.75em

Mientras este tiene un tamaño de 18 píxeles
